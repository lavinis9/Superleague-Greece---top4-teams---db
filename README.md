<p>Το συγκεκριμένο side project αφορά μία ολοκληρωμένη Βάση Δεδομένων με <big>πραγματικά δεδομένα</big> που προέρχονται από την ιστοσελίδα του transfermarkt εφόσον μιλάμε για μία δουλεία των Top 4 όμαδων του ελληνικού πρωταθλήματος ποδοσφαίρου(SuperLeague).</p>

<p>ΚΥΡΙΟ ΘΕΜΑ:
