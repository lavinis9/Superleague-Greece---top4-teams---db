<h3>SuperLeague Top 4 Teams Database</h3>
<hr>
<p>Το συγκεκριμένο side project αφορά μία ολοκληρωμένη Βάση Δεδομένων που υλοποιήσαμε σαν ομάδα εγώ και τα άτομα που καταγράφονται στο τέλος από το README. Για την υλοποίηση χρησιμοποιήσαμε <big>πραγματικά δεδομένα</big> που προέρχονται από το website του transfermarkt εφόσον μιλάμε για μία δουλεία των Top 4 όμαδων του ελληνικού πρωταθλήματος ποδοσφαίρου(SuperLeague).</p>

<p>ΚΥΡΙΟ ΘΕΜΑ:
Καταγραφή των επιδόσεων των τεσσάρων μεγάλων ομάδων (Ολυμπιακός, Παναθηναϊκός, ΑΕΚ, ΠΑΟΚ). Θα αποθηκεύονται στοιχεία για τους παίκτες, τους αγώνες, τα γκολ και άλλα στατιστικά, επιτρέποντας τη σύγκριση της αποτελεσματικότητάς τους.</p>
